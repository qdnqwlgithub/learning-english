# 📊 英语时态 · 语态对照表（主动 & 被动）

| 时态 | 主动语态 | 被动语态 |
| :--- | :--- | :--- |
| **一般现在时** | do / does | am / is / are + done |
| **一般过去时** | did | was / were + done |
| **一般将来时** | will / shall do | will / shall + be done |
| **一般过去将来时** | would / should do | would / should + be done |
| **现在进行时** | am / is / are + doing | am / is / are + being done |
| **过去进行时** | was / were + doing | was / were + being done |
| **将来进行时** | will / shall be doing | will / shall + be being done* |
| **过去将来进行时** | would / should be doing | would / should + be being done* |
| **现在完成时** | have / has + done | have / has + been done |
| **过去完成时** | had + done | had + been done |
| **将来完成时** | will / shall have done | will / shall + have been done |
| **过去将来完成时** | would / should have done | would / should + have been done |

> \* 将来进行时与过去将来进行时的被动形式在语法上成立，但实际使用极少，通常用主动语态或其它结构替代。

## 主动语态时态应用场合

| 时态 | 应用场合 | 例句 |
|------|----------|------|
| **一般现在时** | 1. 经常性、习惯性动作<br>2. 客观事实、普遍真理<br>3. 现在的状态或特征 | He **gets up** at 7 every day.<br>The sun **rises** in the east.<br>She **works** in a bank. |
| **一般过去时** | 1. 过去某个时间发生的动作<br>2. 过去的状态或习惯 | She **finished** her homework yesterday.<br>He **was** a teacher. |
| **一般将来时** | 1. 将来发生的动作或状态<br>2. 预测、计划、意图 | We **will arrive** tomorrow.<br>It **will rain** soon. |
| **一般过去将来时** | 1. 从过去某个时间看将来要发生的动作（常用于宾语从句） | He said he **would come** later. |
| **现在进行时** | 1. 现在正在进行的动作<br>2. 现阶段正在进行的动作（不一定此刻）<br>3. 按计划将要发生的动作（go, come, leave等） | She **is reading** now.<br>I **am learning** English these days.<br>We **are leaving** tomorrow. |
| **过去进行时** | 1. 过去某个时刻正在进行的动作<br>2. 过去某段时间持续进行的动作 | At 8 pm, I **was watching** TV.<br>Yesterday morning, he **was working** in the garden. |
| **将来进行时** | 1. 将来某个时刻正在进行的动作<br>2. 按计划将来一定会发生的动作 | At 10 am tomorrow, I **will be flying** to Paris.<br>They **will be meeting** at the station. |
| **过去将来进行时** | 1. 从过去角度看将来某个时刻正在进行的动作 | He said he **would be waiting** for us. |
| **现在完成时** | 1. 过去发生的动作对现在有影响<br>2. 从过去持续到现在的动作<br>3. 经历、经验 | I **have lost** my key.（现在找不到）<br>She **has lived** here for 10 years.<br>I **have been** to Japan. |
| **过去完成时** | 1. 过去某个时间之前已经完成的动作（过去的过去） | When I arrived, the train **had left**. |
| **将来完成时** | 1. 到将来某个时间为止已经完成的动作 | By next year, I **will have graduated**. |
| **过去将来完成时** | 1. 从过去角度看，到将来某个时间为止已经完成的动作 | He said he **would have finished** by Friday. |

## 被动语态时态应用场合

| 时态 | 应用场合 | 例句 |
|------|----------|------|
| **一般现在时被动** | 强调动作的承受者（经常性、普遍性） | English **is spoken** all over the world. |
| **一般过去时被动** | 过去某个动作的承受者 | The window **was broken** by Tom. |
| **一般将来时被动** | 将来动作的承受者 | The meeting **will be held** tomorrow. |
| **一般过去将来时被动** | 从过去看将来动作的承受者 | He said the work **would be done** soon. |
| **现在进行时被动** | 现在正在被做的动作 | The house **is being painted** now. |
| **过去进行时被动** | 过去某个时刻正在被做的动作 | When I arrived, the car **was being repaired**. |
| **将来进行时被动** | ⚠️ 极少使用，通常避免 | （通常用其他时态代替） |
| **过去将来进行时被动** | ⚠️ 极少使用 | （几乎不用） |
| **现在完成时被动** | 到目前为止已经被完成的动作 | The task **has been finished**. |
| **过去完成时被动** | 过去某个时间之前已经被完成的动作 | The letter **had been sent** before he arrived. |
| **将来完成时被动** | 到将来某个时间为止已经被完成的动作 | The bridge **will have been completed** by next year. |
| **过去将来完成时被动** | 从过去角度看将来某个时间之前已被完成的动作 | He said the project **would have been finished** by then. |

## 快速选择指南

| 你想要表达的意思 | 选哪个时态 |
|----------------|-----------|
| 经常做的事 / 事实 | 一般现在时 |
| 过去发生的具体事 | 一般过去时 |
| 将来要做的事 | 一般将来时 |
| 此刻正在做 | 现在进行时 |
| 过去某刻正在做 | 过去进行时 |
| 将来某刻正在做 | 将来进行时 |
| “已经…了”（有影响） | 现在完成时 |
| “到…时候已经…了”（过去的过去） | 过去完成时 |
| “到将来…时候已经…了” | 将来完成时 |
| 强调“被…” | 用被动语态 |
